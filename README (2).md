# Geospatial Data Analysis (Task 4)

Analyze location-based data to identify optimal areas for business expansion. This project visualizes sales data geographically and detects regions with high demand but low brand presence using automated mapping tools.

## Included Files
* `geospatial_analysis.py`: Main Python execution script.
* `sales_location_data.csv`: Geolocation coordinates, demand metrics, and sales numbers.

## Requirements
To run this analysis script, install the following required packages:
```bash
pip install pandas matplotlib
```

## How to Run
1. Keep both files in the same directory folder.
2. Run the script:
   ```bash
   python geospatial_analysis.py
   ```
